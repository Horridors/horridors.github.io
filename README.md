# Horridors — Web Game

The official site for **Horridors**, hosted at [horridors.com](https://horridors.com).

- `/` — the web game (play in browser)
- `/privacy/` — privacy policy (required for Google Play Store)

## Hosting

This repo is served by **GitHub Pages** at the custom domain `horridors.com`.
The `CNAME` file at the repo root tells GitHub which domain to serve.

## How to update the site

Just push changes to the `main` branch. GitHub Pages auto-deploys within ~1 minute.
